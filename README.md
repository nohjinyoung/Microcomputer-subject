# Microcomputer-subject

## 과목 소개
마이크로컴퓨터 관련 전공 과목에서 수행한 과제 및 텀프로젝트를 정리한 저장소입니다

## 개발 환경
- Language: C
- Board: STM32F407가 탑재된 학교 자체 키트
- IDE: IAR workbench Embedded IDE
